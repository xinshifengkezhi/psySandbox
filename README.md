1.该项目是针对心理沙盘数据集的训练代码，模型基于关系型图神经网络设计，不过数据集的关系类型暂时设为1；
2.原始数据是一个心理沙盘软件生成的txt格式的日志，记录了操作时间，操作类型和操作内容，标签是专家评分0-10分取平均数，分数越低越严重；
3.该项目六个模块：
  3.1.第一个是将txt格式的日志转化为易懂的结构化数据，具体格式为由图id命名的csv文件，一行对应一条操作；
  3.2.第二个是针对结构化的数据的建图过程，并进一步转化为可用于训练时的数据结构，标签进一步分类，将十一个分数转化为正常、轻度、重度三个类别；
    将模具名称和象征语义通过paraphrase-multilingual模型转化为一定维度的文本向量；
    节点特征包括沙具的方向角度、沙具的欧氏坐标、情感得分、正向、中性、负向得分、转化后的名称文本和语义文本；
  3.3.第三个是图的可视化，展示图中各节点的分布情况；
  3.4.第四个是数据的平衡，原始数据的三个类别对应图数量严重不平衡，最大最小相差几十倍，因此数据平衡采用类内随机组合的方式；
    该模块里使用了二进制的思想来随机选取组合方式并确保不会重复；
  3.5.第五个是训练模块，日志包括数据集的基本信息、划分信息、模型的一些信息、测试集的结果和其他一些参数；
    生成内容包括最佳模型权重、完整模型结构、优化器状态、训练初始参数、训练历史；
  3.6.第六个是边重要性解释，绘制环形图
  各模块的参数配置均放在configs里，其中的各个参数在CustomConfig.py里有详细的说明；
4.测试环境Python3.8.10，各个库的版本均放在environment.yaml里，根据情况安装对应版本的库，
  或者直接执行下列代码(这里面有很多用不上的库，谨慎运行)：
 ### Python environment setup with Conda
    conda env create -f environment.yaml
